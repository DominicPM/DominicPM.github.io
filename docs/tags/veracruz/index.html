Accepted IEEE TETCSI special issue paper

I am delighted to announce that the paper Private delegated computations using strong isolation has been accepted into a special issue on advances in emerging privacy-preserving computation at IEEE Transactions on Emerging Topics in Computer Science (TETCSI). New technical report on the Icecap and Veracruz projects

I am delighted to announce the recent release of an Arm Research technical report on the IceCap and Veracruz confidential-computing projects. The report, “Private delegated computations using strong isolation”, is co-authored by Mathias Brossard, Guilhem Bryant, Basma El Gaabouri, Xinxin Fan, Alexandre Ferreira, Edmund Grimley-Evans, Christopher Haster, Evan Johnson, Derek Miller, Fan Mo, Nick Spinale, Eric van Hensbergen, Hugo J. M. Vincent, Shale Xiong, and me.

September 7, 2021

Privacy-preserving collaborative compute with Veracruz

Veracruz is a project that I’ve been working on for a while internally within Arm Research. Recently, Veracruz was released as an open-source project on Github, with all development and design discussions moving into the open. This was a precursor to the project being adopted by the Linux Foundation’s Confidential Computing Consortium (CCC), which I’m pleased to say finally happened a few weeks ago, at the beginning of April.
